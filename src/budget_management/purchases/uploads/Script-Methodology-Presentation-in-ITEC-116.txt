> User Management Module
- Sa User Management Module, base sa interview namin with the former president ng campus, binanggit niya na kailangan ng feature kung saan makikita at mae-edit ng users ang kanilang profile information. Kaya, dinisenyo namin ang registration, login, at password reset functions. Kapag nag-register ang user gamit ang form, required silang mag-login at pagkatapos, automatic silang madi-direct sa dashboard screen. May access din sila sa profile screen kung saan pwede nilang i-update ang kanilang personal information. gumamit kami ng bootstrap as our framework para sa responsive design at madali namin itong mai-customize. Although ang bootstrap ang naging foundation ng aming design, we manually modify the features para mag-match sa mockup na ginawa namin.

> Dashboard Module
- for Dashboard Module, we also design it for the users who can see the overall summary of the important financial data tulad ng financial statements or status, budget allocations, upcoming events, and transaction history ng kanilang expenses. We also use a Bootstrap here for the UI para maging malinis at modern interface siya for the users. We design it with charts and tables para mas madaling makita ang mga data, at para maging user-friendly ang presentation ng financial health ng organization

> Conclusion
- In conclusion, we use incremental model for our system which is the ANGAT SIKAT. In this approach, hinati namin ang development ng system sa iba't ibang stages or increments, kung saan ang bawat increment ay nadadagdag ng specific na functionality tulad ng User Management at Dashboard Module. In every step, we tested and refined the system para ma-improve continuously hanggang mabuo ang buong web-based solution. As per zylei's introduction, we design these system to automate the financial management processes of SASCO, that intention to do the transparent and efficient na pag handle ng budget expenses ng organization. With ANGAT SIKAT, this will help the organizations to monitor their budgets, streamline the approvals, and sa paggamit ng mga data analytics para magkaroon ng mas maayos na system ng financial management system. Our study is focus on solving current challenges, particularly in accountability and monitoring of expenses, and this will expect to give a good system for the student organization. and also, we are currently in build 1 in incremental model. ang mga nagagawa pa lang namin ay ang core functionalities na focus sa Budget Management Module. Kapag natapos na namin ito, magdadagdag kami ng mga notifications at iba pang improvements sa build 2 at sa mga susunod pang builds.
